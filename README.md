## Getting Started

### Prerequisites
* `NodeJS v14`

### Installation

```bash
$ git clone https://github.com/RocketMeet/RocketMeet-mailer.git

$ cd RocketMeet-mailer

$ npm install
```

### Setup

Make a copy of the .env.example file to .env and set the `EMAIL` and `PASSWORD` variables corresponding to your Gmail account.

### Run

```bash
$ npm run dev
```

You will be able to access the API from http://localhost:5000

### Testing

```bash
$ npm run test
```

### Deployment

In your .env, change the `NODE_ENV` to `production` and set the `EMAIL` and `PASSWORD` variables corresponding to your Gmail account.
